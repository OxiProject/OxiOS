# OxiOS

OxiOS is a fast, minimal and stable Linux distribution built on Debian.

Our goal is to create a simple operating system with a modern user experience and powerful tools.

## Core Components

- Oxi Package Manager
- Oxi Driver Manager
- Oxi Updater
- Oxi Game Manager

## Philosophy

- Fast
- Minimal
- Stable
- Easy to use
